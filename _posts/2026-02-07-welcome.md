Welcome to the newly redesigned FVUtils organization website! We've updated the site to provide a better overview of all the tools and utilities available in the FVUtils ecosystem.

## What's New

- **Organized Project Catalog**: All projects are now categorized as either user-facing tools or developer-focused utilities
- **Detailed Descriptions**: Each project now has a comprehensive description with links to documentation
- **Modern Design**: Updated visual design with project cards and improved navigation
- **News Section**: Stay updated with the latest developments across FVUtils projects

## About FVUtils

FVUtils provides a collection of open-source tools for functional verification engineers and developers. From constrained-random stimulus generation with PyVSC to coverage analysis with PyUCIS, transaction tracing with DV Transaction Trace, and HDL integration with PyHDL-IF, FVUtils offers modern, Python-centric solutions for today's verification challenges.

## Get Involved

All FVUtils projects are open source and available on [GitHub](https://github.com/fvutils). We welcome contributions, bug reports, and feature requests. Check out individual project repositories to learn more and get started.

Happy verifying!
